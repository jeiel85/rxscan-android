# Security Policy

RxScan is designed so prescription images, OCR text, medicine selections, and
saved history stay on the user's device.

## Reporting

Do not send real prescription images, OCR dumps, or medicine lists by default.
Open a GitHub issue with non-sensitive reproduction details, or contact the
maintainer privately for a protected disclosure path when sensitive evidence is
unavoidable.

Useful non-sensitive details:

- app version;
- public database version;
- parser/matcher policy version;
- device model and Android version;
- error code;
- synthetic or fully redacted fixture when possible.

## Current status

This repository is at bootstrap stage. The app currently contains no OCR,
medicine matching, public database updater, account system, analytics, ads, or
patient-data backend.
