RxScan Android

Privacy Draft

This page describes the intended privacy posture for the engineering preview. It must be updated to match the actual released build before any public app release.

Current bootstrap build

  • No OCR implementation is active.
  • No prescription image or OCR text upload path exists.
  • No account, analytics, ads, Firebase Analytics, or remote configuration is included.
  • The Android app manifest does not request Internet permission in Goal 00.

Intended production boundaries

  • Prescription images and OCR text remain on device.
  • Official public medicine data is downloaded only as generic signed artifacts.
  • Saved prescription history is optional and encrypted.
  • User support should not request unredacted prescriptions by default.
